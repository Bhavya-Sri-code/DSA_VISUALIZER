# 🚀 DSA Visualizer

An interactive **Data Structures and Algorithms Visualizer** built using **HTML, CSS, and JavaScript**.
This project helps users understand how different algorithms work through **real-time animations**.

---

## 🌟 Features

### 🔢 Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
  📊 Visual representation using animated bars

---

### 🔍 Searching Algorithms

* Linear Search
* Binary Search
  🎯 Step-by-step element highlighting

---

### 📚 Stack Visualizer

* Push operation
* Pop operation
* Overflow & Underflow handling
  📦 Visualized using vertical blocks

---

### 🚶 Queue Visualizer

* Enqueue
* Dequeue
* Front & Rear tracking
  ➡️ Displayed horizontally for clarity

---

### 🌐 Graph Visualizer

* Create nodes and edges
* Breadth First Search (BFS)
* Depth First Search (DFS)
  🔗 Interactive graph traversal animations

---

## 🎨 UI Highlights

* Clean and modern design
* Smooth animations
* Color-coded operations:

  * 🟡 Comparing
  * 🔴 Swapping
  * 🟢 Sorted

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/Bhavya-Sri-code/DSA_VISUALIZER.git
```

2. Open the project folder

3. Run the project:

* Double-click `index.html`
  OR
* Use:

```
start index.html
```
